# 📊 Olist E-Commerce Sales & Customer Analysis

An end-to-end data analytics project that analyzes over **100,000 Brazilian e-commerce orders** from the **Olist marketplace** using **Python, SQL, and Power BI**. The project focuses on uncovering sales trends, customer behavior, seller performance, product insights, and logistics efficiency to support data-driven business decisions.

---

## 📌 Project Objectives

This project aims to answer key business questions such as:

- How are sales performing over time?
- Which product categories generate the highest revenue?
- Who are the most valuable customers?
- Which sellers perform the best?
- What factors influence customer satisfaction?
- How does delivery performance affect review scores?

---

## 📂 Dataset

This project uses the **Brazilian E-commerce Public Dataset by Olist**, containing information on customers, orders, orders_item, products, sellers, payments, reviews, and geolocation.

**Dataset Source:**
[https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce]

> The dataset is not included in this repository due to its size. Please download it from Kaggle before running the project.

## 💡 Key Insights

- Sales showed steady growth throughout the analysis period.
- cama_mesa_banho, beleza_saude, and esporte_lazer were among the top-performing product categories.
- Customer satisfaction was strongly associated with delivery performance.
- Late deliveries generally received lower review scores.
- Most revenue originated from a small number of high-performing sellers.
- Customer purchases were concentrated in major Brazilian states.
