# Dataset

This project uses the **Brazilian E-commerce Public Dataset by Olist**.

## Dataset Source

You can download the dataset from Kaggle:

[https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce]

## Dataset Description

The dataset contains over **100,000 e-commerce orders** placed between 2016 and 2018 on the Olist marketplace. It includes information on:

- Customers
- Orders
- Order Items
- Payments
- Reviews
- Products
- Sellers
- Geolocation
- Product Category Translation
